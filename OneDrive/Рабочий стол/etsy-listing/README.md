# Проект "Etsy Listing"

![Статус сборки](https://img.shields.io/badge/Live-сайт-brightgreen)

Этот проект — страница с товарами Etsy на React.  
Карточки товаров отображаются в сетке 4 на 4, показываются уровни запасов и бейдж для цифровых товаров.

## Прямая ссылка на сайт

[Перейти на сайт](http://localhost:5173/etsy-listing/)  
*(Замените на ваш GitHub Pages URL после деплоя, например `https://username.github.io/etsy-listing/`)*
